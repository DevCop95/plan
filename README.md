# 📊 Plan · Ejecutiva Comercial Yanbal · Cartagena

Sitio web profesional de presentación de plan comercial para ejecutiva de Yanbal en Cartagena. Página de una sola vista con animaciones de revelado y diseño elegante.

## ✨ Características

- Diseño oscuro con paleta dorada y verde esmeralda
- Tipografías: Instrument Serif + Geist Sans
- Animaciones de revelado por scroll
- Iconos con Lucide Icons
- Responsive · sin framework externo
